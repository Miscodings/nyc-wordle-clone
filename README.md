A New York City-themed version of the classic Wordle game. Guess the 5-letter NYC-related word (like `BRONX`, `PIZZA`, or `TRAIN`) in 6 tries.

## Features
* **NYC Vocabulary:** The winning words are all NYC-themed.
* **Dictionary Validation:** Checks if your guess is a real English word before accepting it.
* **Classic Rules:** Green = Correct spot, Yellow = Wrong spot, Gray = Not in word.
* **Animations:** Tiles shake if you enter an invalid word.

## Files
Ensure you have these three files in the same folder:
1. `index.html` (The game board)
2. `style.css` (The design and colors)
3. `script.js` (The logic and word lists)

## How to Run
1. Download the files to a folder on your computer.
2. Double-click **`index.html`** to open it in your web browser (Chrome, Safari, Firefox, etc.).
3. Start typing to play!

> **Note:** You need an internet connection when you load the game so it can fetch the dictionary of valid English words.
